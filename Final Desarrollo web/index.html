Just jump DEMO

Este espacio fue creado con la finalidad de ayudar a toda la industria creativa independiente, dandoles un espacio para difundir sus creaciones, ya sean solo demos, proyectos sin terminar, juegos completos, entre otras cosas; se está comenzado la elaboracion de un foro, con el cual se podran comunicar de una manera mas facil, donde todos puedan compartir sus conocimientos, y asi poderse ayudar el uno al otro.

Demostracion pequeña de un juego 2d de plataforma en estilo runner (como el juego de google chrome cuando no tienes internet), creado en unas horas con el motor de juegos unity, con funciones basicas como salto, muerte, reinicio de juego, tabla de puntuacion, y record con la mejor puntuacion, puede que todavia contenga algunos errores, debido a que es una etapa temprana todavia para el juego.
